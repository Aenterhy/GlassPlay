Glass Play
=========
Unofficial Google Play browser

Features
--------
- Voice command: **Store ...**
- Application ratings, company

How to install
--------------
1. [Download latest APK](https://drive.google.com/file/d/0B4TWd3UiNpfcOEZjbUprd3duNzg/edit?usp=sharing)
3. `adb install` glassplay.apk
